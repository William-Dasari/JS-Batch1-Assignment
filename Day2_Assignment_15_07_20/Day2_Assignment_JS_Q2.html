Remaining methods of Strings and Arrays

Strings :

Day2 Assingnment

1.CharCodeAt() :

Returns the Unicode of the character at the specified index

2.fromCharCode() :

Converts Unicode values to characters

3.includes() :

Checks Wheater a string contains the specified string/characters

4.localeCompare() :

Compares two strings in the current locale
A Number, indicating whether the reference string comes before, after or is the same as the compareString in sort order. Returns one of three values:
-1 if the reference string is sorted before the compareString
0 if the two strings are equal
1 if the reference string is sorted after the compareString

5.repeat :

Returns a new string with a specified number of copies of an existing string

6.replace() :

Searches a string for a specified value, or a regular expression, and returns a new string where the specified values are replaced

7.search() :

Searches a string for a specified value, or regular expression, and returns the position of the match

8.slice() :

Extracts a part of a string and returns a new string

9.starts With() :

Splits a string into an array of substrings

10.substr() :

Extracts the characters from a string, beginning at a specified start position, and through the specified number of character

11.subsrting() :

Extract the character from a string between two specified indices

12.trim():

Returns the primitive value of a String object

2.copyWithin()

4.every() :

Checks if every element in an array pass a test

5.fill() :

Fill the elements in an array with a static value

6.filter() :

Creates a new array with every element in an array that pass a test

7.find():

Returns the value of the first element in an array that pass a test

8.findIndex():

Returns the index of the first element in an array that pass a test
